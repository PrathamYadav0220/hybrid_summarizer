Methodology Overview

Stage 1: Extractive Summarization

BERT-based Analysis

  • Text tokenization and preprocessing
  • BERT N-2 layer embeddings generation
  • K-means clustering of sentence embeddings
  • Selection of sentences closest to centroids

Stage 2: Abstractive Summarization

T5 Model Processing

  • Input format: 'summarize: ' + extracted text
  • T5-base model with 220M parameters
  • Beam search with configurable length control
  • Human-readable summary generation
